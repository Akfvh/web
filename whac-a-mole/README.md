# WHAC-A-MOLE
[유튜브 튜토리얼](https://www.youtube.com/watch?v=ec8vSKJuZTk&t=3896s)을 보고 만든 두더지잡기 게임
## Features
- 각 셀에 랜덤으로 두더지 출현
- 클릭시 점수 +1
- 타이머 끝날 시 alert, 최종 점수 표시
